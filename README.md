Banking Data Analytics Project
Project Overview

This project performs Exploratory Data Analysis (EDA) and business intelligence reporting on a banking dataset.
The goal is to analyze customer financial behavior, understand banking performance, and build interactive dashboards for decision making.

The project combines Python (EDA), SQL, and Power BI to transform raw banking data into meaningful insights.

Objectives
-Analyze customer financial data
-Understand loan and deposit trends
-Identify patterns based on income band and customer demographics
-Build interactive dashboards for business analysis
-Generate insights to support banking strategy and decision making

Tools & Technologies
-Programming & Analysis
-Python
-Pandas
-Matplotlib
-Seaborn
-Database
-SQL Server / SQL
-Visualization
-Power BI

Project Workflow
1 Data Collection
The banking dataset was imported from the database and prepared for analysis.

2 Data Cleaning
-Removed missing values
-Handled inconsistent data
-Converted columns to proper data types

3 Exploratory Data Analysis (EDA)
Using Python:
-Customer demographic analysis
-Loan vs deposit analysis
-Income band distribution
-Transaction behavior patterns

4 Data Visualization
-Charts were created to understand trends such as:
-Customer distribution
-Financial product usage
-Banking growth trends

5 Power BI Dashboard
-An interactive dashboard was built with:
-Total Customers
-Total Loans
-Total Deposits
-Income Band Distribution
-Customer Joining Trends
-Financial Product Analysis

Key Insights
-Some insights generated from the analysis:
-Customers in higher income bands have significantly higher deposit balances.
-Loan usage varies across age groups and income levels.
-Recent years show an increase in customer onboarding.
-Deposit growth is higher compared to loan growth in certain segments.
-These insights can help banks improve customer targeting and financial product strategies.
